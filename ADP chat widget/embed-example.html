Rotorua NZ Chatbot - Embed Demo

This page demonstrates the chatbot widget. Click the red button in the bottom-right corner.

How to embed on any website

Add these two lines before the closing </body> tag:

<!-- Rotorua NZ Chatbot Widget -->
<script src="http://localhost:3000/chatbot-widget.js" data-api-url="http://localhost:3000"></script>

Replace http://localhost:3000 with your server URL when deployed.
